# java-postgresql-university-app
# University Database Management System

Java desktop application integrated with a PostgreSQL database for managing university data, including students, courses, enrollments, professors, and classrooms.

## Tech Stack
- Java
- PostgreSQL
- JDBC
- GUI (Swing / JavaFX)

## Features
- Execute SQL queries through the application
- Manage students, courses, and enrollments
- Display results in a graphical interface
- Support for joins, nested queries, and aggregation queries

## Project Structure
- `/app` – Java application source code
- `/database` – SQL scripts and database-related files
- `project-requirements.pdf` – Assignment requirements
- `er-diagram.pdf` – Database design (ER diagram)
- `project-presentation.pdf` – Project presentation

## How to Run
1. Create the database in PostgreSQL
2. Run the SQL scripts from the `/database` folder
3. Open the Java project from `/app` in IntelliJ IDEA
4. Configure database connection (JDBC)
5. Run the application

## Notes
This project was developed as part of a university assignment and focuses on database integration and application-level interaction.
